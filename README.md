# Retail-Rocket-Analysis

## Project Overview

This project focuses on analyzing user behavior data from an e-commerce platform to identify bottlenecks in the conversion funnel and evaluate customer loyalty.

## Key Insights & Results

* **Conversion Funnel:** Identified a significant drop-off at the "Add to Cart" stage with a **2.6% CR**, which is below the industry average of 6-11%.
* **Overall Conversion:** The end-to-end conversion rate (View to Transaction) stands at **0.8%**.
* **Retention & Loyalty:** Found that **22% of transacting users** are loyal customers (performing more than one transaction).
* **Cart Abandonment:** The abandonment rate is **68%**, aligning with the typical e-commerce range of 70-75%.

## Tech Stack

* **Python:** Data manipulation and analysis.
* **DuckDB:** Efficient SQL querying on a large dataset of **2.7M+ events**.
* **Pandas & Seaborn:** Data cleaning and visualization.

## How to use

1. Download the `events.csv` from the Retail Rocket dataset on Kaggle.
2. Run the `Retail_project.ipynb` notebook to reproduce the analysis.
